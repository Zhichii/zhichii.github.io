# “主意”介绍

我有很多的主意。

在我告诉你我的所有主意之前，我先讲讲为什么我弄了这个板块。

我认识了AI之后，我总是会把我的各种想法跟AI讲。

扥是，我发现，它们并不总是能理解我的意思。它们并没有我想象中那么强大。

再加上谄媚等各种原因，我就直接停用了它们。

（还有一个原因是我不应该依赖它们。）

所以现在，除了测试用途，我都不应该使用它们。

所以，我选择记录下我的想法，并用个人博客（虽然说其实是Docusaurus）这样管理它们。

现在，让我第一次记录我的想法。

虽然这些主意可能看起来幼稚，不过我现在正好是初中二年级。所以中二一点也没什么大不了的。

所以请分享你有用的建议！

我固然不希望有人批评我。但我不得不听取建议。
