# 基于规则的人工智障

~~**请不要骂我😭我害怕**~~

虽然这听起来确实很拉，而且确实实际不可行，但我确实计划开启一项漫长的工程。

即，用手动的，可解释的规则，去手动编写一个人工智能。

这听起来确实就不可能。

但我仍然天真地认为这是可能的。

毕竟，只有三十万个参数的AI都能说话。

关键在于，我们肯定要高可扩展性。这样才能持续更新，并不会退步太多。

我去搜了一下CYC。我确实感到震惊。只不过可惜的是这貌似不是自然语言。

我的理想是，它将继承我对世界的认识。毕竟是我写的。
